Projects

Some of my recent work and personal projects

Context-Aware Medical Assistant using PEFT (LoRA) and RAG

Fine-tuned a medical LLM using LoRA for efficient adaptation, paired with RAG to deliver accurate, context-aware clinical responses grounded in medical literature.

Challenge

Full fine-tuning of large medical LLMs was computationally prohibitive, and generic models lacked domain-specific accuracy for clinical queries.

Solution

Applied Parameter-Efficient Fine-Tuning (LoRA) to adapt the model with minimal compute, and integrated Retrieval-Augmented Generation to ground responses and reduce hallucinations by retrieving verified medical documents.

LoRA RAG LangChain HuggingFace

LLM-Assisted Explainable Anomaly Detection for Smart Homes

Built an XAI pipeline that detects IoT anomalies in smart homes and uses TinyLlama to generate human-readable explanations for non-technical users.

Challenge

Traditional anomaly detectors flag issues but provide no explanation, making it hard for homeowners to understand or act on alerts.

Solution

Combined anomaly detection models with TinyLlama to translate technical detections into plain-English explanations, bridging the gap between ML output and user understanding.

TinyLlama XAI IoT Python

Transformers from Scratch using PyTorch

Implemented the full Transformer architecture from scratch in PyTorch -- including multi-head attention, positional encoding, and encoder-decoder blocks -- to deeply understand the internals.

Challenge

Understanding the math behind scaled dot-product attention and getting the tensor dimensions right across multi-head attention layers was non-trivial.

Solution

Broke down the "Attention Is All You Need" paper step-by-step, implemented each component independently with unit tests, then composed them into the full architecture.

PyTorch Transformers NLP Python

ADHD Prediction from High-Dimensional Neuroimaging Data

Developed ML models to predict ADHD from fMRI brain scans, tackling the curse of dimensionality with feature selection and ensemble methods.

Challenge

Neuroimaging data has thousands of features with limited samples, leading to severe overfitting and poor generalization with standard classifiers.

Solution

Applied dimensionality reduction techniques and robust feature selection pipelines, combined with cross-validated ensemble models to achieve reliable predictions.

Scikit-learn fMRI Feature Selection Python

METAR Time Series Forecasting using FBProphet

Forecasted aviation weather (METAR) data using Facebook Prophet, enabling better flight planning through accurate temperature and visibility predictions.

Challenge

METAR data has irregular reporting intervals, missing values, and multiple seasonal patterns (daily, yearly) that standard models struggle to capture.

Solution

Leveraged FBProphet's additive decomposition to model multiple seasonalities, handled missing data with interpolation, and tuned changepoint sensitivity for accurate forecasts.

FBProphet Pandas Plotly Python ARIMA

Osteoporosis Classification with SAM-Guided Segmentation and Transfer Learning

Classified osteoporosis severity from X-ray images by combining SAM-based bone segmentation with transfer learning to achieve high diagnostic accuracy.

Challenge

Raw X-ray images contain irrelevant background noise, and limited labeled medical datasets make training deep classifiers from scratch unreliable.

Solution

Used Segment Anything Model (SAM) to isolate bone regions, then applied transfer learning with pre-trained CNNs on the segmented images to improve classification accuracy with limited data.

SAM Transfer Learning PyTorch OpenCV
